## What is Beast Card Clash?

**Beast Card Clash** is a turn-based collectible card strategy game built in Godot 4. The game fuses competitive card gameplay with educational themes, drawing inspiration from Colombian native wildlife and the faculties of the Universidad Nacional de Colombia (UNAL).

Each player commands a character — a Colombian native animal representing a UNAL faculty — and uses a deck of elemental cards to outmaneuver up to three opponents in a strategic battle of placement, dice rolling, and card play.

## Key features

<CardGroup cols={2}>
  <Card title="Elemental card system" icon="fire">
    Cards carry elemental types — Air, Earth, Energy, Fire, and Water — creating strategic depth in every round.
  </Card>
  <Card title="Colombian wildlife characters" icon="paw">
    Play as a Bear, Condor, Chameleon, or Frog — each with multiple skins inspired by real Colombian species.
  </Card>
  <Card title="Faculty-based teams" icon="graduation-cap">
    Choose from nine teams representing UNAL faculties: Acetiles, ADN, Ingeniosos Elementales, Photo Agros, Pluma Dorada, RPC Team, Real Pincel, VA Games, and Zootecnicos.
  </Card>
  <Card title="3D dice rolling" icon="dice">
    An interactive 3D dice determines valid rock positions each turn, adding a luck element to tactical play.
  </Card>
  <Card title="AI bot opponents" icon="robot">
    Battle against 1–3 AI bots with auto-generated decks in every session.
  </Card>
  <Card title="Battle state machine" icon="diagram-project">

  </Card>

## Technology

Beast Card Clash is built with:

- **Godot 4.6** (GDScript primary, with optional C# support via .NET 8)
- **GDScript** for gameplay logic, UI, and autoloads
- **C#** for select systems (optional .NET integration)
- **GDShader** for visual effects
- **Dialogue Manager** plugin for in-game narrative
